Definition 6.4.

If \(V\) is a vector space and \(T\in L(V,V)\) then we say that \(\v\in V\wo\set{\0}\) is an eigenvector of \(T\) if \(T\v=\l \v\) for some scalar \(\l\in\F\text{.}\) If \(\v\) is an eigenvector and \(T\v=\l\v\) we say that \(\l\) is the eigenvalue associated to \(\v\text{.}\)
