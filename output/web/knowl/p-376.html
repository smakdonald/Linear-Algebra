The Interaction [cross-reference to target(s) "lem:interaction" missing or not unique] suggests how to find a basis for a vector space \(V\text{:}\) start with a spanning set and remove vectors until one arrives at a minimal spanning set, which will be a basis. Alternatively start with a linearly independent set and add vectors until one arrives at a maximal linearly independent set. However this process involves choices (not all of which will work). So there can be more than one basis for \(V\text{.}\)
