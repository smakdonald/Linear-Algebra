We can rewrite the system as
\begin{equation*} x\cmat[1\\1\\2] + y\cmat[2\\1\\7] + z\cmat[-3\\-2\\-2] = \cmat[-7\\-5\\8]. \end{equation*}
Thus the system is asking which coefficients \(x, y, z\) we can use in a linear combination of three vectors on the left hand side to obtain the vector on the right hand side.
