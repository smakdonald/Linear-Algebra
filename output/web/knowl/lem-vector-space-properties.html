Lemma 2.11.

If \(V\) is a vector space over the field \(\F\) then the definitions we have made work well with our intuitions in the following ways.
  1. All identities are unique. There is a unique element \(0\in \F\) satisfying A3. There is a unique \(1\in \F\) satisfying M3. There is a unique element \(\0\in V\) satisfying VA3.
  2. All inverses are unique. For \(x\in \F\text{,}\) \(y\in\F\wo\set{0}\text{,}\) and \(\v\in V\) there are unique elements \(-x, y\inv \in \F\) and \(-\v\in V\) such that
    \begin{equation*} x+(-x) = 0 \qquad y y\inv = 1 \qquad \v+(-\v) = \0. \end{equation*}
  3. Inverting twice gets you back where you started: for all \(x\in \F\text{,}\) \(y\in \F\wo\set{0}\text{,}\) and \(\v\in V\) we have
    \begin{equation*} -(-x) = x \qquad (y\inv)\inv = y \qquad -(-\v) = \v. \end{equation*}
  4. For all \(x\in \F\) and \(\v\in V\) we have \((-x)\v = -(x\v)\text{.}\) In particular \((-1)\v=-\v\text{.}\)
  5. For \(x, y\in\F\) and \(\v\in V\) we have
    1. \(x y=0\) if and only if at least one of \(x, y\) is \(0\)
    2. \(x\v=\0\) if and only if \(x=0\) or \(\v=\0\text{.}\)
