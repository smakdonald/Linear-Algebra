Definition 2.26.

If \(V\) and \(W\) are vector spaces over a field \(\F\text{,}\) we let \(L(V,W)\) denote the set of all linear maps from \(V\) to \(W\text{,}\) so
\begin{equation*} L(V,W) = \setof{T:V\to W}{\text{$T$ is linear}}. \end{equation*}
We have different notation for the special case of \(L(V,W)\) where \(W=\F\) (which is of course a vector space over \(\F\)). We write \(V\dual \defeq L(V,\F)\text{.}\) These linear maps are especially important because they correspond to the left hand sides of linear equations.
