Suppose that \((V_i)_{i\in I}\) is a family of vector spaces over a field \(\F\text{.}\) We would like to exhibit a vector space \(W\) that contains (copies of) each of the \(V_i\) without assuming that they are subspaces of the same vector field to begin with. Since \(W\)contains the \(V_i\) it must of course also contain linear combinations of vector from the \(V_i\text{.}\) We will arrange that these linear combinations cannot “accidentally” produce any sort of cancellation. In particular for instance we want it to be the case that if \(\v\in V_i\) and \(\w\in V_j\) then the only way to have \(\v+\w=\0\) is for each of \(\v\) and \(\w\) to be \(\0\text{.}\)
