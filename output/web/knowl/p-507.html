For a matrix \(A\) with entries in \(\C\) we define \(A^*=(\overline{A})^T\) to be the conjugate-transpose of \(A\text{.}\) Specifically, if \(A=[a_{ij}]\) then \(A^*=[\overline{a_{ji}}]\text{.}\) Transposeing commutes with conjugation so we could also write \(A^*=(\overline{A^T})\text{.}\)
