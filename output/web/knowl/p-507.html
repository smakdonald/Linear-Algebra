Let \(W=\setof{p . (x^2+1)}{p \in \R[X]}\text{.}\) There is a vector space isomorphism
\begin{equation*} \R[X]/W \cong \C \end{equation*}
We start with the map \(T: \R[X]\to \C, T(p(x))=p(i)\text{,}\) that is given by evaluating a polynomial at the complex number \(i\text{.}\) Now if \(p(X) (x^2+1)\in W\) then we have \(T(p(x)(x^2+1))=p(i) (i^2+1)=0\) since \(i^2=-1\text{.}\) Thus we have shown that \(W\subseteq \ker(T)\text{.}\) By the previous theorem there is a linear map
\begin{equation*} \overline{T}: \R[X]/W\to \C, \qquad\overline{T}( p(x)+W)= p(i). \end{equation*}
I claim this map is a bijection. To see it is surjective, if \(a+bi\in \C\) then we see that \(\overline{T}( a+bx+W)=a+bi\text{.}\) To see that \(\overline{T}\) is injective note that if \(p(x)+W\in \ker(\overline{T})\) then \(p(i)=0\) so \(p(-i)=0\) since the roots of polynomials with real coefficients come in conjugate pairs. Thus \(x^2+1=(x-i)(x+i)\mid p\text{,}\) that is \(p\in W\text{.}\) Therefore \(p+W=W=\0_{V/W}\text{.}\) We see that \(\overline{T}\) is also injective.
