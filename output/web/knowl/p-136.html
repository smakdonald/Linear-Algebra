That leaves only VA3 and VA4. We need to show that \(\0\in W\) and that if \(\w\in W\) then also \(-\w\in W\text{.}\) Both of these work by choosing appropriate scalings. Since \(W\neq \emptyset\) there exists some \(\w\in W\text{,}\) so then \(\0 = 0\w \in W\) (because \(W\) is closed under linear combinations). Similarly, if \(\w\in W\) then \(-\w=(-1)\w\in W\text{.}\)
