The first part is immediate since any linear combination from \(S\) is also a linear combination from \(T\text{,}\) so any linear combination from \(S\) equaling \(\0\) is also a linear combination from \(T\) equaling \(\0\text{,}\) so is, by hypothesis, trivial. For the second part suppose that some vector \(\0\neq \v\in \linspan(S)\) can be represented in two ways:
\begin{equation*} \Slv = \v = \Smuv. \end{equation*}
(We've thrown all the vectors used in either representation into the set \(\set{\vs}\text{.}\)) Subtracting the right hand side from the left we have
\begin{equation*} \sum_{i=1}^n (\l_i-\mu_i) \v_i = \0, \end{equation*}
and thus, since \(S\) is linearly independent, \(\l_i=\mu_i\) for all \(i\text{.}\) Now, eliminating those \(v_i\) for which \(\l_i=0\) from the set \(\set{\vs}\) gives the claimed uniqueness.
