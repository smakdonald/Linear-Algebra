To show that \(U+U\perp\) is all of \(V\text{,}\) start by picking an orthonormal basis \(\set{\us[r]}\) for \(U\text{.}\) Given \(\v\in V\) set
\begin{equation*} \w = \v - \sum_{i=1}^r \ip{\v,\u_i} \u_i. \end{equation*}
Note that \(\u:=\v-\w= \sum_{i=1}^r \ip{\v,\u_i} \u_i \in U\) so it suffices to show that \(\w\in U\perp\text{.}\) For this in turn it suffices to show that \(\ip{\w,\u_j}=0\) for \(j=1,2\dots,r\text{.}\) We have
\begin{equation*} \ip{\w,\u_j} = \ip{\v,\u_j} - \sum_{i=1}^r \ip{\v,\u_i}\ip{\u_i,\u_j} = \ip{\v,\u_j} - \ip{\v,\u_j} = 0. \end{equation*}
We have shown \(\v=\u+\w\) where \(\u\in U\) and \(\w\in U\perp\text{,}\) so \(\v\in U+U\perp\text{.}\) Thus \(V\subseteq U+U\perp\text{.}\) Since the opposite containment is true, we conclude \(V= U+U\perp\text{.}\)
