Suppose \(V,W\) are real vector spaces each equipped with an inner product, and that \(T\in L(V,W)\text{.}\) We define the adjoint of \(T\text{,}\) written \(T^*:W\to V\) as follows. Given a vector \(\w\in W\) we define a linear map in \(V\dual\) by
\begin{equation*} \v \mapsto \ip{T(\v),\w}. \end{equation*}
This is clearly linear (once we check that we can in fact compute \(\ip{T(\v),\w}\) since \(T(\v)\in W\)). Therefore, by the Riesz Representation Theorem, there is a unique vector, that we denote by \(T^*(\w)\text{,}\) with the property that for all \(\v\in V\) we have
\begin{equation*} \ip{T(\v),\w} = \ip{\v,T^*(\w)} \end{equation*}
The next lemma verifies that \(T^*:W\to V\) is linear.
