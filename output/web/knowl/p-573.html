We show uniqueness first: assume there exists a symmetric bilinear function \(B:V\times V\to W\) such that \(Q(\v)=B(\v,\v)\text{.}\) Then we have, for \(\v,\w\in V\text{,}\)
\begin{equation*} Q(\v+\w) = B(\v+\w,\v+\w) = B(\v,\v) + 2B(\v,\w) + B(\w,\w) = Q(\v) + 2B(\v,\w) + Q(\w). \end{equation*}
Hence
\begin{equation*} B(\v,\w) = \frac{Q(\v+\w)-Q(\v)-Q(\w)}2 \end{equation*}
is determined by the values of \(Q\text{.}\)
