We prove the result by induction on \(\abs{I}\text{.}\) If \(\abs{I}=0\) then \(I=\emptyset\) and we can take \(T=S\text{.}\) Otherwise \(\abs{I}>0\text{.}\) Pick \(\v\in I\) and set \(I' = I\wo\set{\v}\text{.}\) By induction there exists a spanning set \(T'\) with \(I' \of T' \of I'\cup S\) with \(\abs{T'}\le \abs{S}\text{.}\) If \(\v\in T'\) then there is nothing further to do; we can take \(T=T'\text{.}\) Otherwise, our plan now is to insert \(\v\) into \(T'\) and kick out one element of \(T'\text{,}\) but not an element of \(I\), so that the result is still spanning. Since \(T'\) is spanning we have some \(\vs\in T'\text{,}\) \(\ls\in \F\) such that
\begin{equation*} \v = \Slv. \end{equation*}
I claim that for some \(\v_i\in T'\wo I'\) we have \(\l_i\not =0\text{,}\) for otherwise \(\v-\Slv=\0\) would be a non-trivial linear combination of vectors from \(I\) equalling \(\0\text{,}\) a contradiction to the linearly independence of \(I\text{.}\) Now note that
\begin{equation*} \v_i = \frac1{\l_i}\parens[\Big]{\v - \sum_{j\not=i} \l_j\v_j} \in \linspan(T' \cup \set{\v} \wo \set{\v_i}), \end{equation*}
so
\begin{equation*} \linspan(T' \cup \set{\v} \wo \set{\v_i}) = \linspan(T'\cup\set{\v}) = V. \end{equation*}
Thus we can take \(T= T' \cup \set{\v} \wo \set{\v_i}\text{.}\) It is spanning, and it is easy to check that it satisfies the other conditions.
