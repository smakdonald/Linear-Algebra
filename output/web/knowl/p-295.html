Now we show that \(B\) spans \(V\text{.}\) Given \(\v\in V\) we can represent \(T(\v)\) as a linear combination \(\sum_1^\ell \mu_j \w_j\) of the \(\w_j\text{.}\) Now \(\v-\sum_1^\ell \mu_j\v_j\) is in \(\ker(T)\) since
\begin{equation*} T\parens[\Big]{\v-\sum_1^\ell \mu_j\v_j} = T(\v) - \sum_1^\ell \mu_j \w_j = \0. \end{equation*}
Having shown that \(\v-\sum_1^\ell \mu_j\v_j\) is in \(\ker(T)\) we get, for some scalars \(\ls[k]\text{,}\) that
\begin{equation*} \v-\sum_1^\ell \mu_j\v_j = \sum_1^k \l_i \u_i \qquad\text{so}\qquad \v = \sum_1^k \l_i \u_i + \sum_1^\ell \mu_j\v_j, \end{equation*}
and \(B\) spans \(V\text{.}\)
