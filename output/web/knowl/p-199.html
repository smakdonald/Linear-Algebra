Consider the matrix \(A\in M_{3\times 3}(\R)\) given by
\begin{equation*} A = \cmat[1&2&-3\\ 1&1&-2 \\ 2&7&-2]. \end{equation*}
Define a function \(T: \R^3 \to \R^3\) by
\begin{equation*} \label{eqn:T} T((x, y, z)) = A \, \mat[x\\ y\\ z] = \cmat[1&2&-3\\ 1&1&-2 \\ 2&7&-2] \, \mat[x\\ y\\ z]= \cmat[x+2y-3z\\ x+y-2z\\ 2x+7y-2z]. \end{equation*}
Standard facts about matrix multiplication tell us that this is a linear map:
\begin{equation*} \begin{aligned} T((x, y, z)+(x', y', z')) &= A \parens[\Bigg]{ \mat[x\\ y\\ z] + \mat[x'\\ y'\\ z']} = A \mat[x\\ y\\ z] + A \mat[x'\\ y'\\ z'] = T((x, y, z)) + T((x', y', z')) \\ T( \l (x, y, z)) &= A \parens[\Bigg]{ \l \mat[x\\ y\\ z]} = \l\, A \mat[x\\ y\\ z] = \l \, T((x, y, z)) . \end{aligned} \end{equation*}
Not only that, but this linear map can be viewed as being constructed using linear combinations. We have
\begin{equation*} T((x, y, z)) = \cmat[1&2&-3\\ 1&1&-2 \\ 2&7&-2] \, \mat[x\\ y\\ z]= \cmat[x+2y-3z\\ x+y-2z\\ 2x+7y-2z] = x\cmat[1\\1\\2] + y\cmat[2\\1\\7] + z\cmat[-3\\-2\\-2]. \end{equation*}
The result of computing \(T((x, y, z))\) is the linear combination, using coefficients \(x, y\text{,}\) and \(z\text{,}\) of the columns of \(A\text{.}\)
