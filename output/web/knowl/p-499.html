It follows from above that we can define the operations in \(V/W\) alternatively by
\begin{equation*} (\u+W) + (\v+W) \defeq (\u+\v)+W \qquad \l(\u + W) \defeq \l\u + W. \end{equation*}
The cost of this is that since many choices of \(\u\) define that same affine subspace \(\u+W\) one needs to check, for instance, that if \(\u+W = \u'+W\) and \(\v+W = \v'+W\) then \((\u+\v)+W = (\u'+\v')+W\text{.}\) We are spared this task. It is easy however to see that our definition agrees with theirs since, using our definition of \(+\text{,}\)
\begin{equation*} (\u+W) + (\v+W) = \u+W+\v+W = \u+\v+W+W = (\u+\v)+W. \end{equation*}
A similar argument proves that the two different definitions of \(\l A\) agree.
