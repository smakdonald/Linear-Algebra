For b), let \(S\of V\) be spanning. By induction on \(\abs{S}\) it is easy to prove that \(S\) contains a minimal spanning set. [If \(\abs{S}=0\) then \(S\) itself is clearly minimal; if \(\abs{S}>0\) then either \(S\) is minimal or it contains a strict subset \(S'\subsetneqq S\) that is spanning. By induction \(S'\) contains a minimal spanning set, which is a minimal spanning set contained in \(S\text{.}\)] Minimal spanning sets are bases by the Interaction Lemma.
