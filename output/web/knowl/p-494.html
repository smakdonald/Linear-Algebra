If \(\ip{,}\) is an inner product on \(V\) and we fix \(\v\in V\) then the map \(\w\mapsto \ip{\v,\w}\) is an element of \(V\dual\) (since in fact this is true for any bilinear form). More is true though; in fact any function in \(V\dual\) can be represented this way. This is the content of the following theorem.
