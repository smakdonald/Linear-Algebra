The result is trivially true for \(\dim(V)=1\text{.}\) Suppose then that \(\dim(V)>1\text{.}\) Let \(\v_1\) be an eigenvector of \(T\) with eigenvalue \(\l\) and set \(W=\linspan\set{\v_1}\text{.}\) Consider the map
\begin{equation*} \begin{aligned} \tilde{T}\colon V/W &\to V/W \\ \v+W &\mapsto T\v+W \end{aligned} \end{equation*}
This map is well defined since whenever \(\v-\v'\in W\) there is some \(\mu \in \C\) such that \(\v-\v'=\mu\v_1\text{.}\) Thus
\begin{equation*} T\v - T\v' = T(\v-\v') = T(\mu\v_1) = \mu T\v_1 = \mu\l \v_1 \in W, \end{equation*}
thus \(T\v+W = T\v'+W\text{.}\) Pick \(\set{\v_2,\v_3,\dots,\v_n}\) such that \(\set{\v_2+W,\v_3+W,\dots,\v_n+W}\) is a basis for \(V/W\) with respect to which the basis of \(\tilde{T}\) is upper triangular. Now consider the basis \(\set{\v_1,\v_2,\dots,\v_n}\) of \(V\text{.}\) With respect to this basis the matrix of \(\tilde{T}\) with respect to \(\set{\v_2+W,\v_3+W,\dots,\v_n+W}\) appears in the lower right hand corner. This is because if
\begin{equation*} \begin{aligned} T\v_j + W = \tilde{T}(\v_j+W) &= \sum_{i=2}^n a_{ij} (\v_i+W) = \left(\sum_{i=2}^n a_{ij} \v_i \right) + W\\ \intertext{then for some $a_{1j}\in \C$ we have} T\v_j &= a_{1j}\v_1 + \sum_{i=2}^n a_{ij} \v_i. \end{aligned} \end{equation*}
Since the first column of the matrix only has a non-zero entry in the first row and, by assumption, the matrix of \(\tilde{T}\) is upper triangular, we have that the matrix of \(T\) is upper triangular.
