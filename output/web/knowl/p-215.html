Now suppose \(S\) spans \(V\) then
\begin{equation*} \begin{aligned} V &=\linspan{S} &\text{ because }S\text{ spans } V\\ &\subseteq \linspan{T} \subseteq \linspan{V} &\text{ because }S\of T\of V\\ &=V &\text{ because }V\text{ spans } V. \end{aligned} \end{equation*}
This shows that \(\linspan(T)=V\text{.}\)
