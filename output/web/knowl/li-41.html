Item 2.

If \(A\in M_{m\by n}(\F)\) then we can define a linear map \(M_A:\F^n\to \F^m\) by doing matrix multiplication. To be completely explicit we define, for \(\v=(v_1,v_2,\dots,v_n)\in \F^n\text{,}\)
\begin{equation*} M_A(\v) = A\v = \parens[\Big]{\sum_{j=1}^n A_{i j} v_j }_{i=1}^m. \end{equation*}
It is a fact, that we will prove soon, that every linear map from \(\F^n\) to \(\F^m\) is of the form \(M_A\) for some matrix \(A\in M_{m\by n}(\F)\text{.}\)
