Thus the problem of solving a Lights Out game with starting position \(M\) is exactly that of finding a set \(S\) if positions such that
\begin{equation*} M + \sum_{(i, j)\in S} M_{i j} = \0, \end{equation*}
where \(\0\) denotes the \(5\times 5\) zero matrix. Since \(a=-a\) for all \(a\in F_2\text{,}\) \(M=-M\) for all \(M\) in \(M_{5\by 5}(\F_2)\text{,}\) the displayed equation above is the same as
\begin{equation*} M = \sum_{(i, j)\in S} M_{i j}, \end{equation*}
that is, representing \(M\) as a \(\Z_2\)-linear combination from \(\setof{M_{i j}}{i, j\in \{1,\ldots, 5\}}\text{.}\) This problem can be solved easily computationally since, as vector spaces, there is no essential difference between \(M_{5\by 5}(\F_2)\) and \(\F_2^{25}\) (we will soon say more rigorously that \(M_{5\by 5}(\F_2)\) and \(\F_2^{25}\) are isomorphic vector spaces). All the algorithmic techniques, such as Gaussian elimination, discussed in a first linear algebra course work equally well over any field.
