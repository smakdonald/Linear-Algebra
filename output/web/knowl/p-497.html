The proof so far has justified that the rule \(f\mapsto \v_f\) defines a function \(\varphi:V^*\to V\) and that this function is given by
\begin{equation*} \varphi(f)=\v_f=\sum_1^n\overline{f(\v_i)} \v_i. \end{equation*}
Now let's try to define an inverse to this function. Consider a function \(\psi:V\to V^*\) given by \(\psi(\sum \l_i \v_i)\) is the unique linear function \(g\in V^*\) so that \(g(\v_i)=\overline{\l_i}\text{.}\)
