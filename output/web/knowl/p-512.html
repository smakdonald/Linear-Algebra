Consider the linear map \(T:\R^2\to \R\) given by \(T(x,y)=x\text{.}\) Then \(\ker(T)=\{(0,y)\mid y\in \R\}\) is the \(y\)-axis. The set \(V/\ker(T)\) is the set of all lineas in the plane parallel to the \(y\)-axis (vertical lines). The range of \(T\) can be identified with the \(x\)-axis.
