In \(\R^3\) the vectors \((1,0, 0), (2,1,0)\) and \((3,2,1)\) are linearly independent. Indeed,
\begin{equation*} \l_1(1,0, 0)+\l_2(2,1,0)+\l_3(3,2,1)=\0 \end{equation*}
if and only if
\begin{equation*} \systeme{\l_1+2\l_2+\l_3= 0, \l_2+2\l_3 =0, \l_3=0 } \end{equation*}
which yields \(\l_1=\l_2=\l_3=0\text{.}\)
