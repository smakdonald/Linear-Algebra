The set \(S=\setof{\cos(x)^n}{n\in \N} \of \R^\R\) is linearly independent. A linear combination from \(S\) equaling the \(0\) function would be a finite sequence \(\l_0,\ls\in \R\) such that
\begin{equation*} \l_0 + \l_1\cos(x) + \dots + \l_n \cos^n(x) = 0 \end{equation*}
for all \(x\in \R\text{.}\) In other words the polynomial \(p = \sum_{i=0}^n \l_iX^i \in \R[X]\) would satisfy \(p(\cos(x))=0\) for all \(x\in \R\text{.}\) Thus every value of \(\cos(x)\) would have to be a root of \(p\text{.}\) We know that \(\cos\) takes infinitely many values and the only polynomial with infinitely many roots is the \(0\) polynomial, so our original linear combination must have been trivial.
