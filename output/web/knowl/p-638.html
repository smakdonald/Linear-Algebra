Often people want to solve linear equations that can't be solved. For instance given a large collection of points \(\set{(x_i,y_i)}_{i=1}^n\) they want to find a straight line that goes through the points. Suppose the straight line has equation \(y=\alpha x+\beta\) for some real numbers \(\alpha, \beta\text{.}\) Then we want to solve for \(\alpha,\beta\) in the equations
\begin{equation*} \begin{aligned} \begin{cases} \alpha x_1 + \beta &= y_1 \\ \alpha x_2 + \beta &= y_2 \\ &\vdots \\ \alpha x_n + \beta &= y_n. \end{cases} \end{aligned} \end{equation*}
If we write \(M\) for the matrix
\begin{equation*} M= \mat[x_1&1\\x_2&1\\\vdots&\vdots\\x_N&1] \end{equation*}
we can write the system above in matrix form as given below
\begin{equation*} \label{eq:regression} M\begin{bmatrix} \alpha \\ \beta \end{bmatrix}= \begin{bmatrix} y_1\\ \vdots \\ y_n \end{bmatrix}. \end{equation*}
