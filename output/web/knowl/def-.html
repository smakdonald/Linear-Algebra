Definition 6.10.

Given any \(T\in L(V,V)\) and any polynomial \(p\in \F[X], p(x)=a_0+a_1X+a_2X^2+\cdots a_dX^d\) we can sensibly define \(p(T)\in L(V,V)\) in the following way: for \(n\geq 0\) set \(T^n=T\circ T\circ \cdots \circ T\) (\(n\) times) and define
\begin{equation*} p(T)=a_0+a_1T+a_2T^2+\cdots a_dT^d. \end{equation*}
