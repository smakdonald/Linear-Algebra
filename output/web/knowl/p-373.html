Let \(I\) be a maximal linearly independent subset of \(V\) and let \(\v\in V\text{.}\) If \(\v\in I\) then certainly \(\v\in \linspan(I)\text{.}\) Otherwise \(T=I\cup \set{\v}\) is a strict superset of \(I\) and is therefore not linearly independent. So there is some non-trivial linear combination of vectors from \(I\cup \set{\v}\) that is \(\0\text{.}\) It cannot be that all the vectors involved in this linear combination come from \(I\text{,}\) else \(I\) would not be linearly independent. Thus for some \(\l\not=0\) and some \(\vs\in I\text{,}\) \(\ls\in \F\) we have
\begin{equation*} \l\v + \Slv = \0 \end{equation*}
and hence
\begin{equation*} \v = -\frac1\l \Slv \in \linspan(I). \end{equation*}
For the second part suppose that \(T\) is a minimal spanning set and \(\Slv = \0\) is a non-trivial linear combination from \(T\) with, without loss of generality, \(\l_1\not=0\text{.}\) Then
\begin{equation*} \v_1 = -\frac1{\l_1} \sum_{i=2}^n \v_i \in \linspan(T\wo\set{\v_1}) \end{equation*}
and hence (by part [cross-reference to target(s) "part:SUT" missing or not unique]) of the spanning lemma applied to \(S=T\wo\set{\v_1}\))
\begin{equation*} \linspan(T\wo\set{\v_1}) = \linspan(T) = V. \end{equation*}
This contradicts the minimality of \(T\text{,}\) so no such linear combination can exist.
