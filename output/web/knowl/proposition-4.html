Proposition 6.9.

Given a vector space \(V\) over a field \(\F\text{,}\) the set
\begin{equation*} L(V,V)=\{T:V\to V \mid T \text{ linear} \} \end{equation*}
endowed with the operations of addition \((T+S)(\v)=T(\v)+S(\v)\text{,}\) scalar multiplication \((\l T)(\v)=\l T(\v)\text{,}\) and composition \((T\circ S)(\v)=T(S(\v))\) (written from now on as multiplication \(TS\)) is a \(\F\)-algebra meaning that it satisfies:
  1. \(L(V,V)\) with its addition and scalar multiplication is a \(\F\)-vector space
  2. Composition is associative: \(A (B C) = (AB)C\) for all \(A, B,C\in L(V,V)\)
  3. Composition distributes over addition: \(A(B+C) = AB+AC\) and \((A+B)C = AC+BC\) for all \(A,B,C\in L(V,V)\)
  4. Composition respects scalar multiplication: \((\l A)(\mu B) = (\l\mu) AB\) for all \(\l,\mu\in \F\text{,}\) \(A,B\in L(V,V)\text{.}\)
