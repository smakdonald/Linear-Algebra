For example, consider the following spanning set for \(\R^3\text{:}\)
\begin{equation*} S=\{(1,4,5), (2,2,0), (3,0,0), (5,2,0)\} \end{equation*}
The following subsets are still spanning (in fact minimal spanning) sets for \(\R^3\)
\begin{equation*} \{(1,4,5), (2,2,0), (3,0,0)\},\quad \{(1,4,5), (2,2,0), (5,2,0)\}, \quad \{(1,4,5), (3,0,0), (5,2,0)\}. \end{equation*}
Thus all of the above sets are bases for \(\R^3\text{.}\) However the subset \(S'=\{(2,2,0), (3,0,0), (5,2,0)\}\) of \(S\) is not a spanning set for \(\R^3\) since any vector in \(\linspan(S')\) has the last coordinate equal to 0.
