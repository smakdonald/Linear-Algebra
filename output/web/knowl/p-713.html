If there are non-trivial linear combinations of the \(\v_i\) that are zero take one, say \(\sum_1^k \mu_i \v_i = \0\text{,}\) with the fewest possible non-zero coefficients. Without loss of generality we may suppose that \(\mu_1\not=0\text{.}\) Now apply the linear map \(T-\l_1I\text{.}\) We have
\begin{equation*} \0 = (T-\l_1I) \sum_1^k \mu_i \v_i = \sum_1^k \mu_i(\l_i-\l_1) \v_i. \end{equation*}
This linear combination of the \(\v_i\) has fewer non-zero coefficients than the original (since the coefficient of \(\v_1\) is now \(0\)) so all the coefficients must be \(0\text{.}\) But then all of the original coefficients must have been \(0\) except \(\mu_1\text{.}\) But \(\mu_1\v_1\not= \0\) since \(\mu_1\not=0\) and \(\v_1\not=\0\) (it is an eigenvector). This contradiction proves that the \(\v_i\) are linearly independent.
