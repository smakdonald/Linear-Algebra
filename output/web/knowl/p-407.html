If \(B\) is a basis for the vector space \(V\) over \(\F\) and \(W\) is another vector space over \(\F\) then linear maps in \(L(V,W)\) are determined uniquely by their values on \(B\text{.}\) To be precise, given an arbitrary function  1  \(f:B\to W\) there exists a unique linear map \(T:V\to W\) such that \(T(\b)=f(\b)\) for all \(\b\in B\text{.}\)
