Suppose \(V,W\) are finite dimensional vector spaces over \(\F\) and \(T:V\to W\) is linear. Given (ordered) bases \(B = \set{\vs}, B'=\set{\ws[m]}\) for \(V\) and \(W\) respectively then the matrix of \(T\) with respect to \(B,B'\) is the matrix \([T]_B^{B'}=A = [a_{ij}]_{i=1}^m\vphantom{[]}_{j=1}^n\) such that for all \(i\in [n]\) we have
\begin{equation*} T(\v_i) = \sum_1^m a_{ji} \w_j. \end{equation*}
Note that this determines the \(a_{ij}\) since \(T(\v_i)\) has a unique expression as a linear combination from \(B'\text{.}\) Note that once we've picked the basis \(B\) every vector in \(v\) can be thought of as specified by its coefficients with respect to \(B\text{.}\) If \(\v=\Slv\) then we can think of \(\v\) as corresponding to the column vector \(\col{\l_1\\\l_2\\\vdots\\\l_n}\text{.}\) Similarly the image \(T(\v)\) can be thought of a a column vector \(\col{\mu_1\\\mu_2\\\vdots\\\mu_m}\) if \(T(\v) = \sum_1^m \mu_j \w_j\text{.}\) The matrix \(A\) is chosen so that
\begin{equation*} \col{\mu_1\\\mu_2\\\vdots\\\mu_m} = A \col{\l_1\\\l_2\\\vdots\\\l_n}. \end{equation*}
