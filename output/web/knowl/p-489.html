Of course typically it is not possible to solve [cross-reference to target(s) "eq:regression" missing or not unique], meaning that this system has no solution \(\alpha, \beta\) because the points we started with are not actually on any line. So instead we wish to find the nearest thing in terms of minimizing distances. More formally, let's consider the actual \(y\)-coordinates of the \(n\) points on some line which correspond to \(x_1, \ldots, x_n\text{.}\) Say these \(y\)-values are \(p_1,\ldots, p_n\) and set \(\vv{p}=\begin{bmatrix} p_1\\ \vdots\\ p_n\end{bmatrix}\text{.}\) The vectors we can get in this way are exactly \(\vv{p}=\alpha\x+\beta = M\begin{bmatrix} \alpha \\ \beta \end{bmatrix}\text{.}\) We are interested in minimizing the distance from \(\vv{y}\) to \(\vv{p}\text{,}\) that is \(\norm{\vv{y}-\vv{p}}\text{.}\) This is equivalent to minimizing \(\norm{\vv{y}-\vv{p}}^2\text{.}\)
