Example 2.25.

  1. \(T:\R^3\to \R\) given by \(T((x, y, z)) = 3x+2y+z\text{.}\)
  2. If \(A\in M_{m\by n}(\F)\) then we can define a linear map \(M_A:\F^n\to \F^m\) by doing matrix multiplication. To be completely explicit we define, for \(\v=(v_1,v_2,\dots,v_n)\in \F^n\text{,}\)
    \begin{equation*} M_A(\v) = A\v = \parens[\Big]{\sum_{j=1}^n A_{i j} v_j }_{i=1}^m. \end{equation*}
    It is a fact, that we will prove soon, that every linear map from \(\F^n\) to \(\F^m\) is of the form \(M_A\) for some matrix \(A\in M_{m\by n}(\F)\text{.}\)
  3. Let \(C[0,1]=\{f:[0,1]\to\R : f\) is continuous\(\}\text{.}\) Define \(I:C[0,1]\to \R\) by \(\displaystyle I(f) = \int_{0}^1 f(x) \, dx\text{.}\) Then \(I\) is linear by standard facts about integration.
