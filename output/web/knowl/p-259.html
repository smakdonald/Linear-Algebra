The uniqueness is clear since if \(\v\in V\) is written as a linear combination from \(B\text{,}\) so \(\v =\sum_1^n\l_i \b_i\text{,}\) then we must have that
\begin{equation*} \label{eq:Tdef} T(\v) = \sum_1^n \l_i T(\b_i) = \sum_1^n \l_i f(\b_i). \end{equation*}
Thus to prove existence we simply need to show that ([cross-reference to target(s) "eq:Tdef" missing or not unique]) defines a linear map. Consider \(\v,\w\in V\) and \(\l,\mu\in \F\text{.}\) We'll need to express \(\v,\w\) as linear combinations from \(B\text{;}\) let \(\set{\vecs{\b}{n}}\) be collection of all vectors from \(B\) that appear in either representation. Suppose
\begin{equation*} \v = \sum_1^n \l_i \b_i \qquad \w = \sum_1^n \mu_i \b_i. \end{equation*}
[Notice that some of the \(\l_i,\mu_i\) might be \(0\text{.}\)] Now we have
\begin{equation*} \l\v + \mu\w = \sum_1^n (\l\l_i+\mu\mu_i) \b_i, \end{equation*}
and this is the unique representation of \(\l\v+\mu \w\) using the basis \(B\text{,}\) except that some of the \(\l\l_i+\mu\mu_i\) might be \(0\text{.}\) Thus
\begin{equation*} T(\l\v+\mu \w) = \sum_1^n (\l\l_i+\mu\mu_i) f(\b_i) = \l\sum_1^n \l_i f(\b_i) + \mu\sum_1^n \mu_i f(\b_i) = \l T(\v)+\mu T(\w). \end{equation*}
