    Section 2.2 Vector Spaces

    Definition 2.9.

    A vector space \(V\) over a field \(\F\) is a collection of objects (that we think of as vectors) such that we can form linear combinations \(x_1\v_1 + x_2\v_2 + \dots +x_n\v_n\) where the vectors \(\v_1,\v_2,\dots,\v_n\) are from \(V\) and the scalars \(x_1,x_2,\dots,x_n\) are from the field \(\F\text{.}\)
    A vector space \(V\) over a field \(\F\) is a set \(V\) together with operations of vector addition \(+\colon V\times V \to V\) and scalar multiplication \(\F\times V\to V\) (the latter of which we denote by simple concatenation; \(x\v\) is the scalar multiple of \(\v\) by \(x\)). These must satisfy the following conditions:
    1. Vector addition is commutative: for all \(\v,\w\in V\) we have \(\v+\w=\w+\v\)
    2. Vector addition is associative: for all \(\u,\v,\w\in V\) we have \(\u+(\v+\w) = (\u+\v)+\w\)
    3. There is a (vector) additive identity: there exists a special vector \(\0\in V\) such that for all \(\v\in V\) we have \(\v+\0=\v\text{.}\)
    4. There are (vector) additive inverses: for all \(\v\in V\) there exists a vector \(-\v\) such that \(\v+(-\v)=\0\)
    1. Scalar multiplication respects field addition: for all \(x, y\in\F\) and \(\v\in V\) we have \((x+y)\v=x\v+y\v\)
    2. Scalar multiplication respects field multiplication: for all \(x ,y\in \F\) and \(\v\in V\) we have \((x y)\v=x(y\v)\)
    3. Scalar multiplication respects the field identity: for all \(\v\in V\) we have \(1\v=\v\)
    4. Scalar multiplication respects vector addition: for all \(x\in \F\) and \(\v,\w\in W\) we have \(x(\v+\w) = x\v+x\w\text{.}\)

    Definition 2.10.

    We make a standard abbreviation in the context of a vector space as follows:
    \begin{equation*} \v-\w \defeq \v+(-\w) \end{equation*}
    for \(\v,\w\in V\text{.}\)

    Lemma 2.11.

    If \(V\) is a vector space over the field \(\F\) then the definitions we have made work well with our intuitions in the following ways.
    1. All identities are unique. There is a unique element \(0\in \F\) satisfying A3. There is a unique \(1\in \F\) satisfying M3. There is a unique element \(\0\in V\) satisfying VA3.
    2. All inverses are unique. For \(x\in \F\text{,}\) \(y\in\F\wo\set{0}\text{,}\) and \(\v\in V\) there are unique elements \(-x, y\inv \in \F\) and \(-\v\in V\) such that
      \begin{equation*} x+(-x) = 0 \qquad y y\inv = 1 \qquad \v+(-\v) = \0. \end{equation*}
    3. Inverting twice gets you back where you started: for all \(x\in \F\text{,}\) \(y\in \F\wo\set{0}\text{,}\) and \(\v\in V\) we have
      \begin{equation*} -(-x) = x \qquad (y\inv)\inv = y \qquad -(-\v) = \v. \end{equation*}
    4. For all \(x\in \F\) and \(\v\in V\) we have \((-x)\v = -(x\v)\text{.}\) In particular \((-1)\v=-\v\text{.}\)
    5. For \(x, y\in\F\) and \(\v\in V\) we have
      1. \(x y=0\) if and only if at least one of \(x, y\) is \(0\)
      2. \(x\v=\0\) if and only if \(x=0\) or \(\v=\0\text{.}\)

    Proof.

    …of these is traditionally considered good for the soul. Some hints: if \(0,0'\in \F\) both satisfy A3 then what is \(0+0'\text{?}\) If \(\w\) and \(\w'\) satisfy \(\v+\w = \v+\w' = \0\) what is \(\w+\v+\w'\text{?}\)

    Example 2.12.

    1. The classic vector spaces (over \(\R\)) are \(\R^2\) and \(\R^3\text{.}\) With identical checks we see that also \(\R^n\text{,}\) \(n\ge 1\) and \(\R^\N\) are vector spaces.
    2. Also vital are subspaces of \(\R^n\text{,}\) such as the plane \(\setof{(x, y, z)}{x+y+z=0}\of \R^3\) and the line \(\setof{t(1,0,-1)}{t\in \R}\of \R^3\text{.}\)
    3. Important remark: note that both the line and the plane given here pass through the origin \((0,0,0)\text{.}\) A line or plane that does not pass through the origin is not a subspace. Why not?
    4. Similarly \(\Q, \Q^2, \Q^3,\dots\text{,}\) and \(\Q^\N\) are vector spaces over \(\Q\) and \(\F^n\) (for any field \(\F\) and any \(n\ge 0\)) is a vector space over \(\F\text{.}\)
    5. The set of polynomial \(\R[X]\) is a vector space over \(\R\) and in fact an \(\R\)-algebra.  2 
    6. \(\R\) is a vector space over \(\Q\)
    7. \(\C\) is a vector space over \(\R\)
    8. The set \(M_{m\by n}(\F)\) of \(m\by n\) matrices whose entries come from \(\F\) is a vector space of \(\F\text{,}\) with the usual notions of adding matrices and multiplying them by scalars.
    9. The set \(C(\R)=\setof{f:[0,1]\to\R}{\text{$f$ is continuous}}\) is a vector space over \(\R\text{.}\) It is a subspace of \(\R^{[0,1]}\text{.}\)
    Many of the most important examples of vector spaces come from taking a restricted collection of vectors from some larger vector space.

    Definition 2.13.

    If \(V\) is a vector space over \(\F\) and \(W\of V\) then we say that \(W\) is a (linear) subspace of \(V\) if \(W\) is non-empty and \(W\) is closed under taking linear combinations. That is to say, for all \(\vs \in W\text{,}\) \(\ls \in \F\) we have \(\sum_{i=1}^n \l_i\v_i \in W\text{.}\)

    Lemma 2.14.

    If \(V\) is a vector space then \(W\) is a subspace of \(V\) if and only if \(\0\in W\) and for all \(\v,\w\in W\text{,}\) \(\l,\mu\in \F\) we have \(\l\v+\mu\w\in W\text{.}\)

    Proof.

    Suppose \(W\) is a subspace of \(V\text{.}\) Since \(W\) is nonempty there exists \(\v\in W\text{.}\) Setting \(\v_1=v,\v_2=-v\) and \(\l_1=\l_2=1\) we deduce that \(\0=v+(-v)\in W\text{.}\) Furthermore setting \(n=2\text{,}\) \(\v_1=v, \v_2=\w\) and \(\l_1=\l, \l_2=\mu\) in the definition of subspace gives \(\l\v+\mu\w\in W\text{.}\)
    Conversely, suppose we know \(\0\in W\) and for all \(\v,\w\in W\text{,}\) \(\l,\mu\in \F\) we have \(\l\v+\mu\w\in W\text{.}\) Then one can prove by induction on \(n\) that \(\sum_{i=1}^n \l_i\v_i \in W\text{.}\)
    The base case \(n=1\) follows by setting \(\l=\l_1,\v=\v_1\) and \(\w=\0\text{.}\) For the inductive step, assuming \(\sum_{i=1}^n \l_i\v_i \in W\) we can set \(v=\sum_{i=1}^n \l_i\v_i\text{,}\) \(\l_1=1\) and \(\mu=\l_{n+1}, w=\v_{n+1}\) to conclude \(\sum_{i=1}^{n+1} \l_i\v_i \in W\text{.}\)

    Lemma 2.15.

    If \(V\) is a vector space over \(\F\) and \(W\of V\) is a subspace of \(V\) then \(W\) is a vector space in its own right (with the operations that come with \(V\text{;}\) we add and scale vectors in \(W\) just as we do in \(V\text{.}\))

    Proof.

    Most of the vector space axioms are satisfied automatically: VA1, VA2, and SM1-4 work in \(W\) because they work for every vector in \(V\) and every scalar in \(\F\text{.}\) One key point is that because \(W\) is closed under taking linear combinations there really is a vector addition map \(+\colon W\times W \to W\) and a scalar multiplication map \(\F\times W\to W\text{.}\) The result of adding two vectors in \(W\) is again a vector in \(W\text{,}\) similarly for scaling.
    That leaves only VA3 and VA4. We need to show that \(\0\in W\) and that if \(\w\in W\) then also \(-\w\in W\text{.}\) Both of these work by choosing appropriate scalings. Since \(W\neq \emptyset\) there exists some \(\w\in W\text{,}\) so then \(\0 = 0\w \in W\) (because \(W\) is closed under linear combinations). Similarly, if \(\w\in W\) then \(-\w=(-1)\w\in W\text{.}\)

    Definition 2.16.

    If \(V\) is a vector space over \(\F\) and \(S\of V\) then the span of \(S\), denoted \(\linspan(S)\text{,}\) is the set
    \begin{equation*} \linspan(S) = \setof[\Big]{\sum_{i=0}^n \l_i\v_i}{\vs\in S, \ls\in \F, n\ge 0}. \end{equation*}
    I.e., it is the collection of all linear combinations of vectors in \(S\text{.}\)

    Example 2.17.

    If \(S=\emptyset\) then \(\linspan(S) =\{\0\}\) since the empty linear combination is by convention the \(\0\) vector.

    Example 2.18.

    If \(S=\{(1,0,-1)\}\subset \R^3\) then \(\linspan(S) =\{\l(1,0,-1) \mid \l\in\R\}\text{.}\) The set of points \(\linspan(S)\) is a line through the origin of \(\R^3\text{.}\)

    Example 2.19.

    \(\R[X]= \linspan\left(\{1, X, X^2, X^3, \ldots, X^i, \ldots \mid i\geq 0\}\right).\) The set of polynomials of degree at most \(d\) is \(\R[X]_{\leq d}= \linspan\left(\{1, X, X^2, X^3, \ldots, X^d\}\right).\)

    Lemma 2.20.

    If \(V\) is a vector space over \(\F\) and \(S\of V\) then \(\linspan(S)\) is a subspace of \(V\text{.}\)

    Proof.

    Clearly \(\0\in \linspan(S)\text{;}\) simply take \(n=0\) in the definition. Now if \(\v,\v'\in \linspan(S)\) and \(\l,\mu\in \F\) we have, for some \(n, n'\ge 0\) and \(\vs\text{,}\) \(\vps\in S\text{,}\) \(\ls\text{,}\) \(\lps\in \F\text{,}\)
    \begin{equation*} \v = \sum_{i=1}^n \l_i \v_i \qquad \text{and} \qquad \v' = \sum_{i=1}^{n'} \l'_i \v'_i. \end{equation*}
    Thus we have
    \begin{equation*} \l\v+\mu\v' = \l\sum_{i=1}^n \l_i \v_i + \mu \sum_{i=1}^{n'} \l'_i \v'_i \in \linspan(S). \end{equation*}

    Example 2.21.

    Example [cross-reference to target(s) "ex:R[X]span" missing or not unique] implies that the polynomials of degree at most \(d\) form a subspace of \(\R[X]\text{.}\)

    Remark 2.22.

    By contrast, the set of polynomials of degree exactly \(d\) do not form a subspace of \(\R[X]\text{.}\) Why not?
    An algebra is a vector space for which we can also multiply two vectors (in addition to multiplying vectors with scalars.
