    Section 5.4 Inner Product Spaces and Orthogonality

    All the vector spaces in this section will be vector spaces over one of the fields \(\R\) or \(\C\text{.}\)
    We will use the complex conjugation operation on \(\C\) given by
    \begin{equation*} \overline{a+bi}=a-bi \text{ for }a,b\in\R \end{equation*}
    and the absolute value function
    \begin{equation*} |a+bi|=\sqrt{a^2+b^2}=\sqrt{(a+bi)(\overline{a+bi})}. \end{equation*}
    Note that \(\overline{a}=a\) for any \(a\in\R\text{.}\) In fact for \(z\in \C\) we have \(z=\overline{z}\) if and only if \(z\in\R\text{.}\)
    An inner product on vector space \(V\) is a function \(\langle - , -\rangle: V\times V\to \F\) which is
    1. conjugate-symmetric: for all \(\u,\v\in V\) we have \(\langle \u,\v\rangle =\overline{\langle \v,\u \rangle}\)
    2. linear in the first entry: for all \(\u,\u', \v\in V\) and \(\l,\mu\in \F\) we have
      \begin{equation*} \langle\l \u+\mu \u',\v\rangle =\l \langle \u,\v\rangle+\mu\langle \u',\v\rangle. \end{equation*}
    3. positive definite for all \(\v\in V\) such that \(\v\neq \0\) we have \(\langle \v,\v\rangle>0\)
    The vector space \(V\) together with its inner product \(\ip{,}\) is called an inner product space. If \(V\) is an inner product space over the field of real numbers \(\R\) we call \(V\) a Euclidean space.
    Suppose \(V\) is an inner product space and denote the inner product of \(\v,\w\in V\) by \(\ip{\v,\w}\text{.}\) The corresponding quadratic function can be used as a way of measuring lengths of vectors in \(V\text{.}\) We define the norm of a vector \(\v\in V\) to be
    \begin{equation*} \norm{\v} = \sqrt{\ip{\v,\v}}. \end{equation*}
    Note that since the inner product is positive definite we have \(\norm{\v}\geq 0\) and moreover \(\norm{\v}>0\) for all \(\v\not=\0\text{.}\)
    For all \(\u,\v,\v'\in V\) and \(\l\in \F\) we have
    \(\langle \v,\0\rangle =\langle \0, \v \rangle=0\)
    \(\langle \v,\u+\u'\rangle= \langle \v,\u\rangle+\langle \v,\u'\rangle\)
    \(\langle \v,\l \u\rangle=\overline{\l}\langle \v,\u\rangle\)
    \(\langle \v,\v\rangle \in\R\)
    \(\norm{\l\v}=|\l|\norm{\v}\text{.}\)
    Exercise.
    If \(\F=\R\) Properties (b) and (c) of an inner product imply that it is a bilinear function. Thus when \(V\) is a Euclidean space, the inner product is a symmetric bilinear function.
    If \(\F=\R\) or \(\C\text{,}\) the dot product \(\langle \v,\w\rangle=v_1\overline{w_1}+v_2\overline{w_2}+\cdots +v_n\overline{w_n}\) is the standard inner product on \(\F^n\text{.}\) For this inner product \(\norm{\v}\) is just the usual length of \(\v\text{.}\)
    The following is an inner product on \(\mathcal{C}[0,1]\text{,}\) the real vector space of continuous functions \(f:[0,1]\to\R\text{:}\)
    \begin{equation*} \ip{p,q} = \definite p(x)q(x) dx{0}1. \end{equation*}
    Checking the conditions is straightforward.
    • Symmetry means \(\ip{p,q} =\ip{q,p}\) for all \(p,q\in \mathcal{C}[-1,1]\text{.}\) (Since \(\F=\R\) we don't need to conjugate.)
      This holds since multiplication of real numbers is commutative so that \(p(x)q(x)= q(x)p(x)\) for all \(x\in [0,1]\) and therefore \(\definite p(x)q(x) dx{0}1= \definite q(x)p(x) dx{0}1\text{,}\) that is \(\ip{p,q} =\ip{q,p}\text{.}\)
    • Linearity means \(\ip{\l p+\mu p',q}=\l \ip{p,q} +\mu \ip{p',q}\)
      \begin{equation*} \begin{aligned} \ip{\l p+\mu p',q} &= \definite \left( \l p(x)+\mu p'(x)\right)q(x) dx{0}1\\ &= \definite \l p(x)q(x)+\mu p'(x)q(x) dx{0}1 & \text{ by distributivity in }\R\\ &= \l \definite p(x)q(x) dx{-1}1 + \mu \definite p'(x)q(x) dx{0}1 & \text{ by linearity of the integral }\\ &= \l \ip{p,q} +\mu \ip{p',q}. \end{aligned} \end{equation*}
    • Positive definiteness means that \(\ip{p,p} >0\) whenever \(p\neq \0\text{.}\)
      This is a consequence of the fact that \(p(x)^2\geq 0\) and so \(\ip{p,p}=\definite p(x)^2 dx{-1}1 \geq 0\) for all \(p\in\mathcal{C}[-1,1]\text{.}\) Moreover, if \(p\neq \0\) there exists \(x\in[0,1]\) so that \(p(x)^2>0\) and in fact by continuity \(p(x)^2>0\) on some interval within \([0,1]\text{.}\) This gives that \(\ip{p,p}=\definite p(x)^2 dx{-1}1>0\) whenever \(p\neq \0\text{.}\)
    If \(\u,\v\) are elements of an inner product space \(V\) we say that \(\u\) and \(\v\) are othogonal (to each other) if \(\langle \u,\v\rangle =0\text{.}\)
    If \(V\) is an inner product space then a list of vectors \(\v_1, \ldots, \v_n\in V\) is a orthonormal list (ONL) if
    \begin{equation*} \langle \v_i, \v_j \rangle =\begin{cases} 0 & \text{ if } i\neq j\\ 1 & \text{ if } i= j \end{cases} \end{equation*}
    An orthonormal basis (ONB) is an orthonormal list which is also a basis for \(V\text{.}\)
    If \(\F=\R\) or \(\C\) then the standard basis \(\e_1,\ldots, \e_n\) of \(\F^n\) is a ONB with respect to the standard inner product.
    Any finite dimensional inner product space \(V\) has an ONB. Moreover a ONB can be obtained using the following algorithm:
    Start with any basis \(\{\b_1,\ldots, \b_n\}\) of \(V\)
    Set \(\v_1=\b_1/\norm{\b_1}\text{.}\)
    For i from 1 to \(n-1\) set
    \begin{equation*} \begin{aligned} \label{GS} \hat{\v}_{i+1} &=&\b_{i+1}-\langle \b_{i+1},\v_1\rangle\cdot \v_1-\cdots -\langle \b_{i+1},\v_i\rangle\cdot \v_i\\ \v_{i+1} &=& \hat{\v}_{i+1} /\norm{\hat{\v}_{i+1} } \nonumber . \end{aligned} \end{equation*}
    Then \(\{\v_1,\ldots, \v_n\}\) is a ONB for \(V\text{.}\)
    We prove by induction on \(i\geq 1\) that \(\v_1,\ldots, \v_i\) is a ONL that satisfies \(\linspan\{\v_1,\ldots, \v_i\}=\linspan\{\b_1,\ldots, \b_i\}\text{.}\)
    Base case: \(i=1\) since \(\v_1=\b_1/\norm{\b_1}\) we deduce that \(\linspan\{\v_1\}=\linspan\{\b_1\}\text{.}\) Moreover
    \begin{equation*} \langle\v_1, \v_1 \rangle =\langle \b_1/\norm{\b_1}, \b_1/\norm{\b_1} \rangle =\frac{1}{\norm{\b_1}^2} \langle \b_1, \b_1\rangle =1. \end{equation*}
    Induction step: assume that \(\v_1,\ldots, \v_i\) is a ONL that satisfies \(\linspan\{\v_1,\ldots, \v_i\}=\linspan\{\b_1,\ldots, \b_i\}\text{.}\) Therefore \(\v_1,\ldots, \v_i\in \linspan\{\b_1,\ldots, \b_i\}\text{.}\) The definition of \(\v_{i+1}\) shows that \(\v_{i+1}\in \linspan\{\v_1,\ldots, \v_i, \b_{i+1}\}=\linspan\{\b_1,\ldots, \b_i,\b_{i+1}\}\text{.}\) Since \(\v_1,\ldots, \v_{i+1}\in \linspan\{\b_1,\ldots, \b_i, \b_{i+1}\}\) we conclude that \(\linspan\{\v_1,\ldots, \v_{i+1}\} \subseteq \linspan\{\b_1,\ldots, \b_{i+1}\}\text{.}\)
    Conversely the inductive hypothesis gives \(\b_1,\ldots, \b_i\in \linspan\{\v_1,\ldots, \v_i\}\text{.}\) Moreover solving for \(\b_{i+1}\) from the equation defining \(\hat{v}_{i+1}\) shows that \(\b_{i+1}\in \linspan\{\v_1,\ldots, \v_i, \v_{i+1}\}\text{.}\) Since \(\b_1,\ldots, \b_{i+1}\in \linspan\{\v_1,\ldots, \v_i, \v_{i+1}\}\) we conclude that \(\linspan\{\b_1,\ldots, \b_{i+1}\} \subseteq \linspan\{\v_1,\ldots, \v_{i+1}\}\text{.}\)
    We may now conclude that \(\linspan\{\v_1,\ldots, \v_{i+1}\} = \linspan\{\b_1,\ldots, \b_{i+1}\}\text{.}\)
    To show \(\v_1,\ldots, \v_{i+1}\) is a ONL we compute by the inductive hypothesis that
    \begin{equation*} \langle \v_\ell,\v_j\rangle=\begin{cases} 0 & \text{if } \ell\neq j\\ 1 & \text{if } \ell = j. \end{cases} \end{equation*}
    for \(1\leq \ell,j \leq i\text{.}\) Now for \(1\leq j\leq i\) we have
    \begin{equation*} \begin{aligned} \langle \hat{\v}_{i+1}, \v_j\rangle &=& \langle \b_{i+1}, \v_j\rangle-\langle \b_{i+1},\v_1\rangle \langle\v_1, \v_j\rangle-\cdots -\langle \b_{i+1},\v_i\rangle \langle\v_i, \v_j\rangle \\ &=& \langle \b_{i+1}, \v_j\rangle - \langle \b_{i+1}, \v_j\rangle \langle \v_j,\v_j\rangle = \langle \b_{i+1}, \v_j\rangle- \langle \b_{i+1}, \v_j\rangle\cdot 1\\ &=&0. \end{aligned} \end{equation*}
    Since \(\hat{\v}_{i+1}=\norm{\hat{\v}_{i+1}}\v_{i+1}\) we have
    \begin{equation*} 0=\langle \hat{\v}_{i+1}, \v_j\rangle=\langle \norm{\hat{\v}_{i+1}}\v_{i+1}, \v_j\rangle = \norm{\hat{\v}_{i+1}}\langle \v_{i+1},\v_j\rangle. \end{equation*}
    In view of \(\norm{\hat{\v}_{i+1}} \neq 0\) we conclude \(\langle \v_{i+1},\v_j\rangle=0\text{.}\) Finally, \(\langle \v_{i+1},\v_{i+1}\rangle=1\) by the same argument as in the base case.
    The case \(i=n\) shows that \(\{\v_1,\ldots, \v_n\}\) is a spanning set for \(V\text{.}\) Since \(\{\b_1,\ldots, \b_n\}\) is a basis of \(V\) we know that \(\dim V=n\text{.}\) By this implies that \(\{\v_1,\ldots, \v_n\}\) is a basis for \(V\text{.}\)
    Find a ONB for the Euclidean vector space \(V_2=\{a+bX+cX^2\mid a,b,c \in \R\}\) of polynomials of degree at most two with inner product \(\langle p,q\rangle = \definite p(x)q(x) dx{0}1.\)
    We start with the basis \(\b_1=1, \b_2=X, \b_3=X^2\text{.}\) Then we set
    \begin{equation*} \v_1=\frac{\b_1}{\norm{\b_1}}=1. \end{equation*}
    Further
    \begin{equation*} \hat{\v}_2=\b_2-\langle \b_2, \v_1\rangle \cdot \v_1= X- \definite X dx{0}1 \cdot 1=X-\frac{1}{2}, \end{equation*}
    \begin{equation*} \norm{\hat{\v}_2}= \definite( X-\frac{1}{2})^2dx{0}1=\frac{1}{3} (X-\frac{1}{2})^3\mid_0^1=\frac{1}{12}, \end{equation*}
    \begin{equation*} \v_2=\sqrt{12}(X-\frac{1}{2}). \end{equation*}
    In the next step we have

    Having a ONB presents the advantage that it make it easy to write any vector in \(V\) as a linear combination of the ONB. We can even give a closed formula for the coefficients of such a linear combination.
    If \(\{\v_1,\ldots, \v_n\}\) is a ONB for an inner product space \(V\) and \(\u\in V\) then the unique way to write \(\u\) as a linear combination of the \(\v_i\)s is
    \begin{equation*} \u = \ip{\u,\v_1} \cdot \v_1+\cdots + \ip{\u,\v_n} \cdot \v_n. \end{equation*}
    Since \(\{\v_1,\ldots, \v_n\}\) is a basis, in particular it spans \(V\text{,}\) so there exist \(\l_1,\ldots, \l_n\) such that
    \begin{equation*} \u =\l_1\v_1+\cdots + \l_n\v_n . \end{equation*}
    Substituting gives \(\ip{\u,\v_i}=\l_1\ip{\v_1, \v_i}+\cdots + \l_n\ip{\v_n,\v_i}=\l_i\text{,}\) as claimed.
    Let \(V\) be an inner product space and let \(\u, \v\in V\text{.}\) Then the orthogonal projection of \(\v\) onto \(\u\) is a vector \(\vv{p}\) so that \(\v=\vv{p}+\w\) with \(\vv{p}\in \linspan\{\u\}\) and \(\w\) orthogonal to \(\u\text{.}\)
    The orthogonal projection of \(\u\) onto \(\v\) is unique and is given by
    \begin{equation*} {\proj}_{\u}(\v)=\frac{\langle \u,\v\rangle}{\norm{\u}^2}\cdot\u. \end{equation*}
    If \(V\) is an inner product space and \(U\of V\) is a subspace then we define the orthogonal complement to \(U\) to be the set
    \begin{equation*} U\perp = \setof{\x\in V}{\text{$\ip{\x,\u}=0$ for all $\u\in U$}}. \qedhere \end{equation*}
    The set \(U\perp\) is a subspace of \(V\text{;}\) this is a consequence of the linearity of \(\ip{,}\) in the first input.
    If \(U\) is a subspace of a finite dimensional inner product space \(V\text{,}\) then
    \begin{equation*} V = U\oplus U\perp. \end{equation*}
    To show that \(U+ U\perp\) is a direct sum we use . The fact that \(U\) and \(U\perp\) have trivial intersection is a consequence of the fact that if \(\u \in U\cap U\perp\) then \(\ip{\u,\u} = 0\) and hence \(\u=0\) by positive definiteness.
    To show that \(U+U\perp\) is all of \(V\text{,}\) start by picking an orthonormal basis \(\set{\us[r]}\) for \(U\text{.}\) Given \(\v\in V\) set
    \begin{equation*} \w = \v - \sum_{i=1}^r \ip{\v,\u_i} \u_i. \end{equation*}
    Note that \(\u:=\v-\w= \sum_{i=1}^r \ip{\v,\u_i} \u_i \in U\) so it suffices to show that \(\w\in U\perp\text{.}\) For this in turn it suffices to show that \(\ip{\w,\u_j}=0\) for \(j=1,2\dots,r\text{.}\) We have
    \begin{equation*} \ip{\w,\u_j} = \ip{\v,\u_j} - \sum_{i=1}^r \ip{\v,\u_i}\ip{\u_i,\u_j} = \ip{\v,\u_j} - \ip{\v,\u_j} = 0. \end{equation*}
    We have shown \(\v=\u+\w\) where \(\u\in U\) and \(\w\in U\perp\text{,}\) so \(\v\in U+U\perp\text{.}\) Thus \(V\subseteq U+U\perp\text{.}\) Since the opposite containment is true, we conclude \(V= U+U\perp\text{.}\)
