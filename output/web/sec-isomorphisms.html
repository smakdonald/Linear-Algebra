    Section 3.4 Isomorphisms

    There is a sense in which the vector space \(V_{d}\) of polynomials of degree at most \(d\) and \(\R^{d+1}\) are similar: each polynomial \(c_0+c_1X+\cdots+c_dX^d\) is determined by its coefficient vector \((c_0,c_1, \ldots, c_d)\in \R^{d+1}\text{.}\) More formally, as we have seen in [cross-reference to target(s) "ex:iso" missing or not unique], there is a linear transformation \(T:\R^{d+1}\to V_d\)
    \begin{equation*} T((c_0,\ldots, c_{d}))=c_0+c_1X+\cdots+c_{d}X^{d}. \end{equation*}
    This linear map has the property that for every \(p\in V_d\) there exists a unique \(\vv{c}\in\R^{d+1}\) (the vector of coefficients of \(p\)) so that \(T(\vv{c})=p\text{.}\) This makes \(T\) a bijection.
    A function ‚Äâ2‚Äâ is called a bijection if it is both injective and surjective.
    It is a standard fact that a function \(T:V\to W\) is a bijection if and only if there exists a function \(S:W\to V\) termed the inverse of \(T\) which satisfies \(S(T(\v))=\v\) for all \(\v\in V\) and \(T(S(\w))=\w\) for all \(\w\in W\text{.}\)
    Let \(V\text{,}\) \(W\) be vector spaces over the same field \(\F\) and let \(T:V\to W\) be a linear map. We say that \(T\) is an isomorphism if there exists a linear map \(S:W\to V\) so that \(S(T(\v))=\v\) for all \(\v\in V\) and \(T( S(\w))=\w\) for all \(\w\in W\text{.}\)
    We say that vector spaces \(V\text{,}\) \(W\) are isomorphic and write \(V\cong W\) if there exists an isomorphism \(T:V\to W\) or \(T:W\to V\text{.}\)
    It turns out (it's a small miracle) that a linear function being an isomorphism is the same as being a bijection.
    Let \(V\text{,}\) \(W\) be vector spaces over the same field \(\F\) and let \(T:V\to W\) be a linear map. Then \(T\) is invertible if and only if \(T\) is a bijection.
    By Remark [cross-reference to target(s) "rem:inverse" missing or not unique] \(T:V\to W\) is a bijection if and only if there exists a function \(S:W\to V\) which satisfies \(S(T(\v))=\v\) for all \(\v\in V\) and \(T(S(\w))=\w\) for all \(\w\in W\text{.}\) To conclude that \(T\) is an isomorphism we still have to show that the inverse of \(T\text{,}\) \(S\) is a linear map. Consider \(\u,\w\in W\) and \(\l,\mu\in \F\text{.}\) Then
    \begin{equation*} T(S(\l\u+\mu\w))= \l\u+\mu\w=\l T(S(\u))+ \mu T( S(\w))=T(\l S(\u)+\mu S(\w)). \end{equation*}
    Since \(T\) is injective we conclude that \(S(\l\u+\mu\w)=\l S(\u)+\mu S(\w)\) so \(S\) is a linear map.
    We have seen a lot of example of vector spaces. It turns out that up to isomorphism finite-dimensional vectors spaces are quite simple. Moreover, the isomorphism class of a finite dimensional vector space is completely determined by its dimension.
    If \(V\) is a finite dimensional vector space over \(\F\text{,}\) of dimension \(n\) say, then \(V\iso \F^n\text{.}\)
    Let \(\set{\v_1, \v_2, \dots, \v_{n}}\) be a basis for \(V\text{.}\) Define \(T:\F^n \to V\) by
    \begin{equation*} T((\l_i)_1^n) = \sum_1^n \l_i\v_i. \end{equation*}
    This is clearly a linear map. The kernel of \(T\) is \(\set{\0}\text{,}\) since if \(\sum_1^n \l_i\v_i = \0\) then by the linear independence of the \(\v_i\) all of the scalars must be \(0\text{.}\) Thus \(T\) is injective by [cross-reference to target(s) "thm:injker" missing or not unique]. Similarly \(\range(T)=V\) since every vector in \(V\) can be expressed as a linear combination of the \(\v_i\text{.}\) Thus, by [cross-reference to target(s) "thm:iso" missing or not unique], \(T\) is invertible, meaning exactly that \(V\) and \(\F^n\) are isomorphic.
    If \(V, W\) are finite dimensional vector spaces over the same field \(\F\) then \(V\cong W\) if and only if \(\dim(V)=\dim(W)\text{.}\)
    Homework.
    Earlier we have seen that linear maps between vector spaces are represented by matrices. More formally we have:
    Let \(V\text{,}\) \(W\) be vector spaces over the same field \(\F\) with \(\dim V=n, \dim W=m\text{.}\) Then there is a vector space isomorphism
    \begin{equation*} L(V,W)\cong M_{m\times n}(\F). \end{equation*}
    Proof Idea. Fix bases \(B, B'\) for \(V\) and \(W\) respectively. The map \(\Theta:L(V,W)\to M_{m\times n}(\F)\) that takes a linear map \(T:V\to W\) to the matrix of \(T\) with respect to \(B,B'\) is an isomorphism (check!).
    We can say more about how the isomorphism above interacts with composition of maps.
    Let \(T:U\to V\) and \(S:V\to W\) be linear maps and let \(B, B', B''\) be bases for \(U, V, W\) respectively. Then
    The function \(S\circ T\) given by \((S\circ T)(\u)=S(T(\u))\) is a linear map.
    If \([S]_B^{B'}, [T]_{B'}^{B''}\) are the matrices representing \(S\) and \(T\) with respect to the given bases then
    \begin{equation*} [S\circ T]_{B}^{B''}=[S]_{B'}^{B''} \cdot [T]_{B}^{B'} \end{equation*}
    If you know about rings: the set \(L(V,W)\) is actually a ring with respect to the operations of addition of functions and composition of functions \((S\circ T)(\v)=S(T(\v))\text{.}\) The above lemma says that the map \(T\mapsto [T]_B^{B'}\) defined in the proof of [cross-reference to target(s) "thm:lintransfisomatrices" missing or not unique] is also a ring isomorphism.
    If you know about commutative diagrams: We may put together Remarks [cross-reference to target(s) "thm:isoF^n" missing or not unique] and [cross-reference to target(s) "thm:lintransfisomatrices" missing or not unique] in the following way: fix bases \(B\) for \(V\) and \(B'\) for \(W\) and let \(\phi_V:\F^n\to V\) and \(\phi_W:\F^m\to W\) be the maps constructed as in [cross-reference to target(s) "thm:isoF^n" missing or not unique] based on \(B\) and \(B'\text{.}\) Then the following diagram commutes
    \begin{equation*} \begin{tikzcd} \F^n \arrow{r}{\phi_V} \arrow{d}[swap]{\Theta(T)=[T]_B^{B'}} & V \arrow{d}{T} \\ \F^m \arrow{r}{\phi_W} & W. \end{tikzcd} \end{equation*}
