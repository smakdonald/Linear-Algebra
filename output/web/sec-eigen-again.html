    Section 6.2 Eigenvectors and Eigenvalues

    The most general tool we have to help us understand high powers of linear maps is the of eigenvectors.

    Definition 6.2.

    For a vector space \(V\) we denote \(L(V,V)\) the vector space of linear maps \(T:V\to V\text{.}\) We call such a map a linear operator on \(V\text{.}\)

    Example 6.3.

    The identity map \(I:V\to V\text{,}\) \(I(v)=v, \forall v\in V\) is an element of \(L(V,V)\text{.}\)

    Definition 6.4.

    If \(V\) is a vector space and \(T\in L(V,V)\) then we say that \(\v\in V\wo\set{\0}\) is an eigenvector of \(T\) if \(T\v=\l \v\) for some scalar \(\l\in\F\text{.}\) If \(\v\) is an eigenvector and \(T\v=\l\v\) we say that \(\l\) is the eigenvalue associated to \(\v\text{.}\)

    Example 6.5.

    1. If \(V\) is the vector space of infinitely differentiable functions from \(\R\to \R\) and \(D:V\to V\) mapping \(f\) to \(f'\) is the differentiation map then any exponential function \(f(x) = Ab^x\) is an eigenvector of \(D\) with eigenvalue \(\ln(b)\text{.}\)
    2. Consider the left shift map on \(\R^\N\) defined by
      \begin{equation*} \begin{aligned} L \colon \R^\N &\to \R^\N \\ (a_i)_1^\infty &\mapsto (a_{i+1})_{i=1}^\infty \end{aligned} \end{equation*}
      Again we have that exponential sequences are eigenvectors: if \(a_i = Ab^{i-1}\) then \((a_i)_1^\infty\) is an eigenvector with eigenvalue \(b\text{.}\) Even \(b=0\) gives an eigenvector; \((1,0,0,\dots)\in \ker(L)\) and is thus an eigenvector with eigenvalue \(0\text{.}\)
    The following simple observation is surprisingly important in finding eigenvectors. It shifts the focus of attention away from the eigenvector onto the eigenvalue.

    Lemma 6.6.

    For a linear map \(T\in L(V,V)\text{,}\) the following are equivalent:
    1. \(T\) has an eigenvector with eigenvalue \(\l\in \F\)
    2. \(\displaystyle \ker(T-\l I) \not=\set{\0}\)
    3. \(T-\l I\) is not injective.
    If the vector space \(V\) is finite dimensional then the statements above are further equivalent to
    1. \(T-\l I\) is not injective,
    2. \(T-\l I\) is not surjective.

    Proof.

    We have
    \begin{equation*} T\v = \l\v \quad\iff\quad T\v-\l \v = \0 \quad\iff\quad (T-\l I)\v = \0 \quad\iff\quad \v\in \ker(T-\l I). \end{equation*}
    This shows the equivalence of 1. and 2. For 2. and 3., see . For the equivalence of 3., 4., and 5. see .

    Example 6.7.

    If
    \begin{equation*} A = \begin{bmatrix} 2&1\\4&-1 \end{bmatrix} \end{equation*}
    then the map \(T=M_A:\R^2\to \R^2\) has an eigenvector of eigenvalue \(\l\) exactly if the matrix
    \begin{equation*} A - \l I = \begin{bmatrix} 2-\l&1\\4&-1-\l \end{bmatrix} \end{equation*}
    has rank less than \(2\text{.}\) We have a quick way of determining (this is a pun) whether the matrix \(\begin{bmatrix} a&b\\c&d \end{bmatrix}\) has rank less than \(2\text{.}\) This happens precisely when \(ad-bc=0\text{.}\) Thus \(\l\) is an eigenvalue of \(T\) exactly if
    \begin{equation*} (2-\l)(-1-\l) - 4 = \l^2 - \l -6 = (\l-3)(\l+2) = 0. \end{equation*}
    Thus the eigenvalues of \(T\) are precisely \(3,-2\text{.}\) To find the corresponding eigenvectors we need to find \(\ker(T-\l I)\) for each of \(\l=3,-2\text{.}\) We have
    \begin{equation*} \begin{aligned} \ker(A-3I) &= \ker\begin{bmatrix} -1&1\\4&-4 \end{bmatrix} = \Span\left\{\begin{bmatrix} 1\\1 \end{bmatrix}\right\} \\ \ker(A+2I) &= \ker\begin{bmatrix} 4&1\\4&1 \end{bmatrix} = \Span\left\{\begin{bmatrix} 1\\-4 \end{bmatrix}\right\}. \end{aligned} \end{equation*}
    If we choose the basis \(B=\{\v,\w\}\) where \(\v=\begin{bmatrix} 1\\1 \end{bmatrix}\) and \(\w=\begin{bmatrix} 1\\-4 \end{bmatrix}\) then the matrices of \(T\) and \(T^n\) with respect to this basis are
    \begin{equation*} [T]_B^B= \begin{bmatrix} 3&0\\0&-2 \end{bmatrix} \qquad \text{and} \qquad [T^n]_B^B= \begin{bmatrix} 3^n&0\\0&(-2)^n \end{bmatrix} \end{equation*}
    respectively.

    Example 6.8.

    Consider the right shift map on \(\R^\N\) defined by
    \begin{equation*} \begin{aligned} R \colon \R^\N &\to \R^\N \\ (a_i)_{i=1}^\infty &\mapsto \left(\begin{cases} a_{i-1} & i \ge 2 \\ 0 & i = 1 \end{cases} \right)_{i=1}^\infty \end{aligned} \end{equation*}
    This map has no eigenvectors. First consider \(\l\not=0\) and a solution to \(R\v=\l\v\text{.}\) If \(\v=(v_i)_{i=1}^\infty \not=\0\) then let \(i\) be minimal such \(v_i\not=0\text{.}\) We have
    \begin{equation*} 0 = (R\v)_i = \l v_i \not=0. \end{equation*}
    This contradiction establishes that \(\v=\0\text{,}\) or in other words that \(\l\) is not a eigenvalue of \(R\text{.}\) On the other hand if \(\l=0\) we'd be looking for \(\v\in \ker(R)\text{.}\) It is clear however that \(\ker(R)=\set{\0}\text{;}\) if \(\v=(v_i)_{i=1}^\infty\) has \(v_i\not=0\) then \((R\v)_{i+1} \not= 0\text{.}\)
    To compute eigenvalues it is very helpful to endow the vector space \(L(V,V)\) with extra structure.

    Proposition 6.9.

    Given a vector space \(V\) over a field \(\F\text{,}\) the set
    \begin{equation*} L(V,V)=\{T:V\to V \mid T \text{ linear} \} \end{equation*}
    endowed with the operations of addition \((T+S)(\v)=T(\v)+S(\v)\text{,}\) scalar multiplication \((\l T)(\v)=\l T(\v)\text{,}\) and composition \((T\circ S)(\v)=T(S(\v))\) (written from now on as multiplication \(TS\)) is a \(\F\)-algebra meaning that it satisfies:
    1. \(L(V,V)\) with its addition and scalar multiplication is a \(\F\)-vector space
    2. Composition is associative: \(A (B C) = (AB)C\) for all \(A, B,C\in L(V,V)\)
    3. Composition distributes over addition: \(A(B+C) = AB+AC\) and \((A+B)C = AC+BC\) for all \(A,B,C\in L(V,V)\)
    4. Composition respects scalar multiplication: \((\l A)(\mu B) = (\l\mu) AB\) for all \(\l,\mu\in \F\text{,}\) \(A,B\in L(V,V)\text{.}\)

    Definition 6.10.

    Given any \(T\in L(V,V)\) and any polynomial \(p\in \F[X], p(x)=a_0+a_1X+a_2X^2+\cdots a_dX^d\) we can sensibly define \(p(T)\in L(V,V)\) in the following way: for \(n\geq 0\) set \(T^n=T\circ T\circ \cdots \circ T\) (\(n\) times) and define
    \begin{equation*} p(T)=a_0+a_1T+a_2T^2+\cdots a_dT^d. \end{equation*}
    A very important fact about polynomials in \(\C[X]\) is the following.

    Theorem 6.11. The Fundamental Theorem of Algebra.

    Any monic polynomial \(p(X)\in \C[X]\) of degree \(k\geq 1\) factors into linear factors; there exist \(\l_1, \l_2, \ldots, \l_k\) such that
    \begin{equation*} p(X) = (X -\l_1)(X - \l_2) \cdots (X - \l_k). \end{equation*}
    The following theorem begins to explore the connection between eigenvalues of \(T\) and polynomials satisfied by \(T\text{.}\)

    Theorem 6.12.

    If \(V\) is an \(n\)-dimensional \(\C\)-vector space and \(T\in L(V,V)\) then \(T\) has an eigenvector.

    Proof.

    Since \(\dim L(V,V) = n^2\) the set
    \begin{equation*} \set{I,T,T^2,\dots,T^{n^2}} \end{equation*}
    is linearly dependent. In other words there are scalars \((\mu_i)_{i=0}^k\) with (without loss of generality) \(\mu_k=1\) such that
    \begin{equation*} \sum_{i=0}^k \mu_i T^i = \0 \in L(V,V). \end{equation*}
    Consider then the polynomial \(p = \sum_{i=0}^k \mu_i X^i \in \C[X]\text{.}\) All monic polynomials (of degree at least \(1\)) over \(\C\) factor into linear factors; there exist \(\l_1,\l_2,\dots,\l_k\) such that
    \begin{equation*} p = (X-\l_1)(X-\l_2)\dots(X-\l_k). \end{equation*}
    Thus
    \begin{equation*} p(T) = (T-\l_1 I)(T-\l_2 I)\dots(T-\l_k I) = \0 \end{equation*}
    Now if \(\v\) is any non-zero vector in \(V\) we have \(p(T)\v=\0\text{.}\) Let \(j\) be minimal such that
    \begin{equation*} \w = (T-\l_{j+1} I)(T-\l_{j+2}I)\dots(T-\l_k I) \v \not= \0. \end{equation*}
    (Note that we might have \(j=k\text{,}\) but certainly \(j>0\text{.}\)) Then by the minimality of \(j\) we have
    \begin{equation*} (T-\l_{j}I) \w = \0, \end{equation*}
    so \(\w\) is an eigenvector of \(T\text{,}\) and \(\l_{j}\) is the corresponding eigenvalue.
    It is quite possible, as the following example shows, to have an operator on \(\C^n\) with only one eigenvector (up to scalar multiples).

    Example 6.13.

    Consider the map (whose matrix with respect to the usual basis has \(1\)s on the diagonal immediately above the main diagonal and \(0\)s everywhere else)
    \begin{equation*} \begin{aligned} S \colon \C^n &\to \C^n \\ (a_i)_1^{n} &\mapsto \left( \begin{cases} a_{i+1} & i < n \\ 0 & i=n . \end{cases} \right)_{i=1}^n \end{aligned} \end{equation*}
    The action of \(S\) on the basis vectors \(\e_1,\e_2,\dots,\e_n\) is to map
    \begin{equation*} \e_n\mapsto \e_{n-1} \mapsto \e_{n-2} \mapsto \dots \mapsto \e_2 \mapsto \e_1 \mapsto \0. \end{equation*}
    Clearly \(\e_1=(1,0,\dots,0)\) is an eigenvector with eigenvalue \(0\text{.}\) The only eigenvectors of \(S\) are multiples of \(\e_1\text{.}\) To see this note that, by an argument precisely analogous to that in [cross-reference to target(s) "ex:right_shift" missing or not unique], there are no non-zero eigenvalues. If \(\v=(v_i)_1^n\in \ker(S)\) then \(S\v = \0\) implies \(v_2=v_3=\dots=v_n=0\text{;}\) in other words \(\ker(S)=\linspan(\e_1)\text{.}\)
    We are not able, in light of Example 6.13, to prove that every linear map \(T:V\to V\) on a finite dimensional \(\C\)-vector space is diagonalizable. The closest we can come in general is a situation somewhat similar to that example.

    Theorem 6.14.

    If \(V\) is a finite-dimensional \(\C\)-vector space and \(T\in L(V,V)\) then \(V\) has a basis \(\set{\vs}\) with respect to which the matrix of \(T\) is upper triangular.

    Proof.

    The result is trivially true for \(\dim(V)=1\text{.}\) Suppose then that \(\dim(V)>1\text{.}\) Let \(\v_1\) be an eigenvector of \(T\) with eigenvalue \(\l\) and set \(W=\linspan\set{\v_1}\text{.}\) Consider the map
    \begin{equation*} \begin{aligned} \tilde{T}\colon V/W &\to V/W \\ \v+W &\mapsto T\v+W \end{aligned} \end{equation*}
    This map is well defined since whenever \(\v-\v'\in W\) there is some \(\mu \in \C\) such that \(\v-\v'=\mu\v_1\text{.}\) Thus
    \begin{equation*} T\v - T\v' = T(\v-\v') = T(\mu\v_1) = \mu T\v_1 = \mu\l \v_1 \in W, \end{equation*}
    thus \(T\v+W = T\v'+W\text{.}\) Pick \(\set{\v_2,\v_3,\dots,\v_n}\) such that \(\set{\v_2+W,\v_3+W,\dots,\v_n+W}\) is a basis for \(V/W\) with respect to which the basis of \(\tilde{T}\) is upper triangular. Now consider the basis \(\set{\v_1,\v_2,\dots,\v_n}\) of \(V\text{.}\) With respect to this basis the matrix of \(\tilde{T}\) with respect to \(\set{\v_2+W,\v_3+W,\dots,\v_n+W}\) appears in the lower right hand corner. This is because if
    \begin{equation*} \begin{aligned} T\v_j + W = \tilde{T}(\v_j+W) &= \sum_{i=2}^n a_{ij} (\v_i+W) = \left(\sum_{i=2}^n a_{ij} \v_i \right) + W\\ \intertext{then for some $a_{1j}\in \C$ we have} T\v_j &= a_{1j}\v_1 + \sum_{i=2}^n a_{ij} \v_i. \end{aligned} \end{equation*}
    Since the first column of the matrix only has a non-zero entry in the first row and, by assumption, the matrix of \(\tilde{T}\) is upper triangular, we have that the matrix of \(T\) is upper triangular.

    Theorem 6.15.

    If \(T:V\to V\) is linear and \(\vs[k]\) are eigenvectors with corresponding distinct eigenvalues \(\ls[k]\) then the \(\v_i\) are linearly independent.

    Proof.

    If there are non-trivial linear combinations of the \(\v_i\) that are zero take one, say \(\sum_1^k \mu_i \v_i = \0\text{,}\) with the fewest possible non-zero coefficients. Without loss of generality we may suppose that \(\mu_1\not=0\text{.}\) Now apply the linear map \(T-\l_1I\text{.}\) We have
    \begin{equation*} \0 = (T-\l_1I) \sum_1^k \mu_i \v_i = \sum_1^k \mu_i(\l_i-\l_1) \v_i. \end{equation*}
    This linear combination of the \(\v_i\) has fewer non-zero coefficients than the original (since the coefficient of \(\v_1\) is now \(0\)) so all the coefficients must be \(0\text{.}\) But then all of the original coefficients must have been \(0\) except \(\mu_1\text{.}\) But \(\mu_1\v_1\not= \0\) since \(\mu_1\not=0\) and \(\v_1\not=\0\) (it is an eigenvector). This contradiction proves that the \(\v_i\) are linearly independent.
    The following corollary says that sometimes this is enough to show that \(T\) is diagonalizable.

    Corollary 6.16.

    Suppose that \(T:V\to V\) is linear and that \(V\) has dimensional \(n<\infty\text{.}\) If \(T\) has \(n\) distinct eigenvalues \(\ls\) then \(T\) is diagonalizable.

    Proof.

    Let \(B = \set{\vs}\) where each \(\v_i\) is an eigenvector with eigenvalue \(\l_i\text{.}\) By [cross-reference to target(s) "thm:distinct" missing or not unique] we know \(B\) is linearly independent, and since it has the correct size it must be a basis.
